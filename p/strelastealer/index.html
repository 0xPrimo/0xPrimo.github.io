StrelaStealer:

Jul 23, 2024
2 minute read

Overview

StrelaStealer is a Malware Stealer that that has been out since 2022. it’s a simple malware stealer that steals Outlook and Thunderbird credentials.

Analysis

Looking at the DLL sections, we can see that the entropy of the .data section is high, which indicates that it is packed.

Unpacking

StrelaStealer stores its payload in the .data section, the first 4 bytes is the payload size, followed by 1024 bytes XOR key, and then the encrypted payload.

For unpacking I wrote the following python script:

import pefile
from struct import unpack

def readdword(v):
    return unpack('<I', v)[0]

pe = pefile.PE("311d9f8c68e5661348e97b7e483e7ea1b3fc08863bb3f5f585581ef081058399.dll")

data_section = None
for section in pe.sections:
    if section.Name.strip(b'\x00').decode() == '.data':
        data_section = section
        break

if data_section:
    raw_data = data_section.get_data()
    print("Extracted .data section raw data:")

    payload_size = unpack('<I', raw_data[0:4])[0]
    xor_key = raw_data[4:1028]
    encrypted_data = raw_data[1028:1028+payload_size]

    out = bytearray()
    for i in range(payload_size):
        out.append(encrypted_data[i] ^ xor_key[i % len(xor_key)] ^ 0xc)

    with open('unpakced.bin', 'wb') as file:
        file.write(out)

After decryption the DLL inject itself with the decrypted pe file.
